# Budget model: LeadOps Studio inbound test

Дата: 2026-04-27

## Важно

30–50 тыс ₽/мес — это не точный прогноз Яндекса, а минимальный бюджет для получения первого сигнала по связке:

> объявление → лендинг → старт Telegram-бота → квалифицированный лид.

Для точного прогноза нужен Яндекс Директ Forecast/Budget Planner по конкретным ключам, гео, расписанию и посадке.

## Рыночные ориентиры CPC

По публичным источникам и кейсам:

- Яндекс Директ работает по аукциону; стоимость зависит от конкуренции, качества объявления, CTR, региона и времени показа.
- Москва дороже регионов.
- Стоматология/медицина в Москве — дорогая ниша: конкурентные запросы могут стоить сотни рублей за клик; по отдельным материалам встречается диапазон 100–1500 ₽, часто 500–800 ₽ для дорогих запросов.
- Ремонт/строительство/кухни в Москве — тоже конкурентно: ориентир 50–300+ ₽ за клик, по части источников для ремонта встречается 30–120 ₽, но в Москве бюджет рекомендуют выше.
- B2B/автоматизация/CRM — сильно зависит от интента; грубый ориентир 80–300+ ₽ за клик.
- РСЯ обычно дешевле поиска, но качество ниже: 30–120 ₽ за клик как грубый стартовый диапазон.

Вывод: 30–50 тыс ₽ — это нижняя граница теста. Для Москвы и дорогих ниш это не “полноценная статистика”, а быстрый smoke test.

## Формула расчёта

```
клики = рекламный бюджет / CPC
старты бота = клики × CR лендинг→бот
квалифицированные лиды = старты бота × CR бот→qualified
demo/созвоны = qualified × CR qualified→demo
```

Стартовые допущения:

- CPC Поиск: 120–300 ₽
- CPC РСЯ: 40–120 ₽
- CR лендинг → бот: 3–8%
- CR бот → qualified: 25–40%
- CR qualified → demo/созвон: 30–50%

## Сценарии

### 30 тыс ₽ / месяц

При CPC 120–250 ₽:

- 120–250 кликов;
- 4–20 стартов бота;
- 1–8 qualified лидов;
- 0–4 demo/созвона.

Вывод: годится только для smoke test: есть ли клики, есть ли старты бота, не проваливается ли оффер.

### 50 тыс ₽ / месяц

При CPC 120–250 ₽:

- 200–416 кликов;
- 6–33 старта бота;
- 2–13 qualified лидов;
- 1–6 demo/созвонов.

Вывод: минимально рабочий тест, если не лезть только в самые дорогие стоматологические запросы.

### 80–120 тыс ₽ / месяц

При CPC 120–300 ₽:

- 266–1000 кликов;
- 8–80 стартов бота;
- 2–32 qualified лидов;
- 1–16 demo/созвонов.

Вывод: нормальный тест для Москвы, если первые 30–50 тыс ₽ показывают адекватный CTR и старт бота.

## Особый случай: стоматология Москва

Если идти в дорогие запросы типа “имплантация”, “стоматология Москва”, “эстетическая стоматология”, CPC может быть 300–800+ ₽.

Тогда 50 тыс ₽ даст всего:

- 62–166 кликов;
- 2–13 стартов бота при CR 3–8%;
- статистики мало.

Рекомендация: для первого теста не покупать самые дорогие медицинские запросы. Лучше:

- боль “не успеваете отвечать на заявки”;
- “заявки с сайта в Telegram/CRM”;
- “автоматическая обработка лидов”;
- РСЯ по собственникам/маркетологам;
- отдельная посадка под клиники.

## Сопутствующие расходы

### Обязательные

- Telegram bot: 0 ₽
- Yandex Metrica: 0 ₽
- Домен .ru: ориентир 200–1000 ₽/год
- SSL: обычно 0 ₽, входит в хостинг/конструктор

### Лендинг

Вариант A: Tilda

- 750–1250 ₽/мес ориентир по тарифу;
- быстрее запустить;
- меньше разработки.

Вариант B: статический лендинг на существующей инфраструктуре

- 0–1000 ₽/мес;
- домен + nginx/static hosting;
- больше контроля.

Рекомендация: для smoke test можно использовать статический лендинг, который уже создан в `data/lead-response-demo/index.html`, и не платить Tilda.

### Бот/интеграции

- BotFather: 0 ₽
- Подключение к существующему OpenClaw/скрипту: 0 ₽ инфраструктурно
- Отдельный VPS, если понадобится: 500–1500 ₽/мес
- n8n/Make/Albato, если no-code: 0–3000 ₽/мес на старте

### Аналитика/коллтрекинг

- Yandex Metrica: 0 ₽
- UTM: 0 ₽
- Коллтрекинг: пока не нужен, потому что CTA в Telegram-бота

## Минимальный бюджет запуска

### Вариант “бережный smoke test”

Разово/мес:

- реклама: 30 000 ₽
- домен: 200–1000 ₽/год
- хостинг/static: 0–1000 ₽/мес
- Telegram bot: 0 ₽
- аналитика: 0 ₽

Итого первый месяц: **31–33 тыс ₽** плюс домен.

### Вариант “нормальный минимальный тест”

- реклама: 50 000 ₽
- домен/хостинг/инфра: 1–3 тыс ₽
- запас на сервисы/инструменты: 2–5 тыс ₽

Итого первый месяц: **53–58 тыс ₽**.

### Вариант “Москва, более честная статистика”

- реклама: 80–120 тыс ₽
- домен/хостинг/инфра: 1–5 тыс ₽

Итого: **85–125 тыс ₽**.

## Рекомендация

1. Не начинать с дорогих стоматологических high-intent запросов.
2. Запустить smoke test на 30–50 тыс ₽:
   - горячие B2B/автоматизация/лиды/CRM запросы;
   - РСЯ по боли “теряете заявки”; 
   - отдельные объявления под клиники и ремонт.
3. Если есть старты бота и qualified лиды — поднять до 80–120 тыс ₽.
4. Если нет стартов бота — чинить оффер/лендинг/CTA, а не докидывать бюджет.

## Что нужно сделать для точного прогноза

1. Собрать финальное семантическое ядро 50–150 ключей.
2. Выбрать гео: Москва/МО или вся РФ.
3. Прогнать через прогноз бюджета Яндекс Директа.
4. Разделить кампании:
   - клиники;
   - ремонт/кухни/окна;
   - B2B automation / CRM / лиды;
   - РСЯ.
5. Получить прогноз: показы, клики, CPC, бюджет.
