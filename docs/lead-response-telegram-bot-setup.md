# Telegram bot setup: LeadOps Studio

## Рекомендуемое имя

Name:
LeadOps Studio

Username варианты:

- `LeadOpsStudioBot`
- `LeadOpsRuBot`
- `LeadResponseLabBot`
- `SpeedLeadLabBot`

Display description:

Быстрый первый ответ на заявки: автоответ, квалификация и передача лида в Telegram/CRM.

About:

Помогаем бизнесу не терять входящие заявки: клиент получает ответ за 1–2 минуты, система собирает вводные и передаёт менеджеру готовый лид.

## Команды бота

/start — начать разбор заявки
/demo — посмотреть demo-flow
/audit — разобрать обработку заявок
/contact — связаться с командой

## Welcome message

Здравствуйте! Это LeadOps Studio.

Мы помогаем бизнесу быстрее обрабатывать входящие заявки: клиент получает ответ за 1–2 минуты, система собирает 3–5 вводных и передаёт менеджеру готовый лид в Telegram или CRM.

Можем быстро показать demo-flow или разобрать, где у вас могут теряться заявки.

Что удобно?

1. Посмотреть demo
2. Разобрать мою воронку
3. Обсудить пилот

## Квалификация входящего лида

Вопросы:

1. Какая у вас ниша?
2. Сколько входящих заявок примерно в месяц?
3. Откуда приходят заявки: сайт, реклама, Telegram, WhatsApp, Авито, другое?
4. Как быстро сейчас отвечаете на заявку?
5. Есть CRM? Какая?
6. Что хотите улучшить: скорость ответа, квалификацию, контроль менеджеров, интеграцию с CRM?
7. Удобно ли показать demo на 10–15 минут?

## Эскалация

Лид считается тёплым, если:

- есть 20+ заявок в месяц;
- есть сайт/реклама/мессенджеры;
- есть боль “не успеваем отвечать”;
- готов посмотреть demo;
- спрашивает цену/пилот.

В этом случае агент должен подготовить краткую карточку и спросить Андрея, подключать ли его.

## Как создать через BotFather

1. Открыть Telegram.
2. Найти `@BotFather`.
3. Отправить `/newbot`.
4. Name: `LeadOps Studio`.
5. Username: один из вариантов выше, должен заканчиваться на `bot`.
6. Скопировать bot token.
7. Передать token агенту для подключения.

## Важно про безопасность

Bot token = секрет. Его нельзя публиковать.

Если token утёк — в BotFather выполнить `/revoke` и выдать новый.

## Важно про ограничения

Бот не может первым написать человеку, который не нажал Start.

Для холодного Telegram outreach есть 2 варианта:

1. Писать через отдельный рабочий user-account, а бот использовать как inbound/квалификатор.
2. Писать через сайты/формы/email, а CTA вести в бота.

Рекомендация на старт: бот + формы/публичные контакты. Если Telegram outbound станет нужен — завести отдельный рабочий номер/account.
