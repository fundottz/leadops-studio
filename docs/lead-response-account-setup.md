# Setup временных аккаунтов для outreach

## Нужно создать

### Email

Требования:

- нейтральный, без имени Андрея;
- выглядит как проект/студия;
- можно использовать для холодных писем и ответов;
- желательно с понятной подписью.

Рекомендованные варианты:

1. `leadops.studio@...`
2. `lead.response.lab@...`
3. `leadops.ru@...`
4. `hello.leadops@...`

Провайдер:

- быстрый временный вариант: Gmail/Yandex/Mail;
- более аккуратный вариант позже: домен + почта на домене.

Для первого теста достаточно обычного ящика. Домен можно не покупать до первых пилотов.

### Telegram

Требования:

- нейтральный username;
- аватар/название проекта;
- описание в 1 строку;
- не использовать личный аккаунт Андрея.

Варианты username:

- `@leadops_studio`
- `@leadops_ru`
- `@lead_response_lab`
- `@speedlead_lab`

Название:

LeadOps Studio

Описание:

Быстрый первый ответ на заявки: автоответ, квалификация, Telegram/CRM.

## Минимальная подпись

Команда LeadOps Studio  
Автоматический первый ответ на входящие заявки  
[project_telegram]  
[project_email]

## Что нужно от Андрея

Либо:

1. Сам создать email и Telegram, прислать адрес/username.

Либо:

2. Подтвердить, что можно создавать аккаунты, и дать доступ/номер/резервный email, если сервис потребует подтверждение.

## После создания

Обновить файлы:

- `notes/lead-response-outreach.md`
- `notes/lead-response-approval-brief.md`
- `notes/lead-response-final-package.md`

И заменить плейсхолдеры:

- `[project_email]`
- `[project_telegram]`
- `[demo_link]`
